Pseudo clases Psudo elementos
Definicion: Una pseudoclase es un selector que marca los elementos que están en un estado específico, por ejemplo, los que son el primer elemento de su tipo, o aquellos por los que el cursor les pasa por encima. Definicion: Los pseudo-elementos en CSS son un mecanismo para acceder a partes del HTML que no tienen asociado un nodo en el DOM. Por ejemplo, está claro que una «parte de nuestro documento» podría ser «la primera línea de un párrafo», «la primera letra de un párrafo» o «la selección que ha hecho el usuario». Pero los pseudo-elementos también son una forma de crear contenido antes o después de un elemento de nuestro DOM a través de CSS.
Pseudo clase 1:

:active es una pseudoclase generalmente usada con la etiqueta button, lo que permite esta funcion es resaltar un elemento cuando el usuario le ha dado click, como aumentar su tamaño de teclas o el color del boton.

pseudo elemento 1:

::firs-letter permite seleccionar la primera letra de un elemento de tipo bloque

Pseudo clase 2:

:first-child permite seleccionar el hijo del elemento seleccionado, por ejemplo el primer tag p que este dentro de una etiqueta div, de esta forma se puede siempre seleccionar un elemento hijo y asignarle estilos.

Pseudo elemento 2:

::selection es una etiqueta que permite referenciar las zonas con las que el usuario y ainteractuo o le dio click.

Pseudo clase 3:

:required es una pseudo clase que afecta las etiquetas input, select y text area que tengan el atributo de required configurado. Esta herramienta es util apra asegurarse que un formato tenga informacion valida antes de enviarse.

pseudo elemento 3:

::After es un elemento que permite agregar algo detras del selector que se esta usando, por ejemplo un emoji o una imagen.

Pseudo clase 4:

:focus permite resaltar elementos con los cuales el usuario interactua. Po rejemplo enfoca una etiqueta input si el usuario le da click para interactuar con esta.

Pseudo elemento 4:

::Selection es un elemento que permite seleccionar una porcion de un elemento que el usuario tenga seleccionada en determinaod momento.

Pesudo clase 5:

:hover permite mostrar un pop up cuando el usuario posiciona el mouse sobre un elemento. Util para otorgar informacion adicional sobre la funcionalidad o proposito de un boton.

Pseudo elemnto 5:

::first-letter permite seleccionar la primera letra de un objeto y modificarlo al gusto del programador.
